# Custom NextJS Starter

Custom created NextJS starter with Typescript, ESLint, TailwindCSS, Next-SEO, Globby, and Cloudfare Analytics

This will be a living starter that I will update as it fits my needs.

## Environment Variables

To run this starter you will need to update the following files:

Header.tsx - Add in proper information for site

generate-sitemap.mjs - Proper URL needed for sitemap

robots.txt - proper URL

\_document.tsx - Cloudfare token needed
